# 贡献指南

感谢您对视频转音频应用项目的关注！

## 如何贡献

### 报告问题

如果发现了 bug 或有功能建议，请：

1. 检查 [Issues](https://github.com/your-username/video-to-audio-app/issues) 中是否已有相关问题
2. 如果没有，请创建新的 Issue，包含：
   - 问题描述
   - 重现步骤
   - 预期行为
   - 实际行为
   - 环境信息（操作系统、Node.js 版本等）

### 提交代码

1. Fork 本仓库
2. 创建功能分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 创建 Pull Request

### 代码规范

- 使用 ESLint 进行代码检查
- 遵循现有的代码风格
- 添加必要的注释
- 确保代码可以正常运行

## 开发环境设置

1. Fork 并克隆仓库
2. 安装依赖：`npm install && pnpm install`
3. 确保已安装 FFmpeg
4. 启动开发服务器：`npm start`

## 许可证

通过贡献，您同意您的代码将在 MIT 许可证下发布。
